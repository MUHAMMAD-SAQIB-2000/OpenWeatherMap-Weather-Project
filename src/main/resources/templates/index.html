About Project

This Project works on fetching data from open source APIs of an organization called "openweathermap".
The result fetched is 90 to 100% accurate.
Project's Backend is written in Spring boot for fetching data from API and extracting required data from it.
Project's Front end is made with pure HTML and CSS to display data on webpage.
